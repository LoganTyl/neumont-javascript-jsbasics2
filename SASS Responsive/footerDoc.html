Usage

Footers are typically used to contain information pertaining to the website, including, but not limited to:

  • authorship information
  • copyright information
  • contact information
  • sitemap
  • back to top links
  • related documents

While there can technically be multiple footer tags in a single page, it is highly recommended to have only one at the bottom of the page.

Code

To create a footer, all you need to do is use the <footer> tag.

Examples

<footer>
 <h4>This is a footer element</h4>
</footer>
