Usage

Buttons are used to be clicked on by a user. When clicked, they can activate a method in the backend of code, among other things.

Code

To create a button, use the '.btn_[option]' class in a button element. The various options for buttons include:

  • success
  • warning
  • danger
  • informative
  • monochrome

Examples

<button type="button" class="btn_success">
 Awesome! This is a success button!
</button>


<button type="button" class="btn_warning">
 Watch out. This is a warning button!
</button>


<button type="button" class="btn_danger">
 Watch out! This is a danger button!
</button>


<button type="button" class="btn_informative">
 You should know... This is an informative button!
</button>


<button type="button" class="btn_monochrome">
 Hey. This is a monochrome button.
</button>
