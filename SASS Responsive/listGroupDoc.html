Usage

List groups can be used as a more fancy version of unordered list.

Code

List groups are made using the .list_gpd class within a div. Each element of the list group must have the list_gpd_item class. Each element uses a <p> tag. Elements can be highlighted using the .active class.

Examples

<div class="list_gpd">
 <p class="list_gpd_item">This is a normal list group element</p>
 <p class="list_gpd_item active">This is a highlighted list group element</p>
</div>
