Usage

Alerts are used to signify an important event or something similar to the user. They display text and have preset colors for various scenarios.

Code

To create an alert, use the '.alert_[option]' class. The various options for alerts include:

  • success
  • warning
  • danger
  • informative
  • monochrome

If you want to add a close button, add a div with a alert_close class

Examples

<div class="alert_success">
 <p>Awesome! This is a success alert!</p>
<div>

Awesome! This is a success alert!


<div class="alert_warning">
 <p>Be careful. This is a warning alert!</p>
<div>

Be careful. This is a warning alert!


<div class="alert_danger">
 <p>Watch out! This is a danger alert!</p>
<div>

Watch out! This is a danger alert!


<div class="alert_informative">
 <p>You should know... This is an informative alert!</p>
<div>

You should know... This is an informative alert!


<div class="alert_monochrome">
 <p>Hey. This is a monochrome alert!</p>
<div>

Hey. This is a monochrome alert.


<div class="alert_monochrome">
 <p>This is a closeable alert.</p>
<div>

This is a closeable alert.
