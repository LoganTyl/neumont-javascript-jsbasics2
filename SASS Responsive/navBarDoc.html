Navigation Bars

Back to Home Page

Usage

Navigation bars are elements - typically used in the header - that contain links to other pages of a website.

Code

Navigation bars are made using the <nav> tag with the .navbar class. Within it, use the <ul> tag with the class of .navbar_link_list. For each item in the list, use the .navbar_list_item class. If you want to add a dropdown element, add a div with the .dropdown class. Inside of that, add a button with the .dropButton class. Beneath that, put a div with the .dropList class. Within that, add the desired links.

Examples

<nav class="navbar">
 <ul class="navbar_link_list">
  <li class="navbar_list_item">
   <a href="navBarDoc.html">List Item 1</a>
  </li>
  <li class="navbar_list_item">
;    <a href="navBarDoc.html">List Item 2</a>
  </li>
  <li class="navbar_list_item dropdown">
   <div class="dropdown">
    <button class="dropButton">Dropdown↓</button>
    <div class="dropList">
     <a href="navBarDoc.html">Dropdown Item #1</a>
     <a href="navBarDoc.html">Dropdown Item #2</a>
     <a href="navBarDoc.html">Dropdown Item #3</a>
    </div>
   </div>
  </li>
 </ul>
</nav>
