Usage

Headers usually contain links to navigate the website, as well as the website's logo. While there can be multiple headers, it is common convention to have only one at the top of a page.

Code

To create a header, all you need to do is use the <header> tag.

Examples

<header>
 <h4>This is a header element</h4>
</header>
