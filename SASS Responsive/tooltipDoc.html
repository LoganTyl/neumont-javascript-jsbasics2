Usage

Tooltips are used to display information to the user when an element is hovered over.

Code

To create a tooltip, give a div the .tooltip class. Within it, put a span with a .toolTipContent class.

Examples

<div class="tooltip">
 <p>Hover Over This</p>
 <span class="tooltipContent">This is some cool tooltip text</span>
</div>
