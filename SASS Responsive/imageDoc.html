Usage

Images can display a picture or an animated gif. By default, their size will remain constant when the screen is resized.

Code

Images can be made rounded into a circle via the .img_circular class. They can also be made to be responsive with the .img_resp class.

Examples

<img src="images/flowers.png" class="img_circular">


<img src="images/flowers.png" class="img_resp">
