Usage

There are various type of tags that can be used to stylize your code.

  • The <em> tag is used for emphasized text.
  • The <strong> tag is used for important text.
  • The <del> tag is used for deleted text.
  • The <ins> tag is used for text inserted into a document.
  • The <code> tag is used for computer code.
  • The <samp> tag is used for sample text from a computer program.
  • The <kbd> tag is used for keyboard input.
  • The <var> tag is used for variables.
