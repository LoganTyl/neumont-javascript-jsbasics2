Usage

Forms are used for the user to fill out information, such as name, address, email, etc.

Code

Forms have their own tag (<form>), along with an action that usually takes the user to another page. Usually, inside forms are inputs (<input>). Inputs can have different types, such as text for text inputs, and submit, which submits the form.

Examples

<form action='formDoc.html'>
 <input type="text" name="name">
 <input type="submit" value="submit">
</form>
