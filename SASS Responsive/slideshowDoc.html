Slideshows

Back to Home Page

Usage

Slideshows are used to display a number of images switching between each other.

Code

Slideshows are made with a div with the .slideshow-wrapper class. Inside it, for each image, put a div with the .slideshow-img. Inside that, put the image you want.
For left/right arrows, put a tags with slideshow-prev and slideshow-next classes, respectfully, within the .slideshow-wrapper class.

Examples

<div class="slideshow-wrapper">
 <div class="slideshow-img">
  <img src="images/flowers.png">
 </div>
 <div class="slideshow-img">
  <img src="images/carmen.png">
 </div>
 <div class="slideshow-img">
   <img src="images/flatfriday.png">
 </div>

 <a class="slideshow-prev">≪</a>
 <a class="slideshow-next">≫</a>
</div>
