Usage

Modals are pop up windows that appear on top of the current page to display information.

Code

To create a modal, create a div with the .mdl class. Within it, create another div with the mdl_content class. To close it, create a button within the mdl_content class with a class of mdl_close **Note: The parent class of the close button MUST be the mdl_content div. To make it appear, give a button the mdl_open class.

s

Examples

<button class="mdl_open">Open the Modal</button>

<div class="mdl">
 <div class="mdl_content">
  <p>This is a modal.</p>
  <button class="mdl_close">Close the Modal</button>
 </div>
</div>
